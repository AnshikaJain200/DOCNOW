Developer Information

Welcome to the Developer page! Here you can learn more about the dedicated team behind DocNow and the technologies used in building this innovative health care recommendation system.

Meet the Team

Anshika Jain

Anshika Jain is the lead developer and project manager of DocNow. With a background in computer science and a passion for health technology, Anshika has driven the vision and development of this project. She specializes in Python, Flask, and front-end technologies.

Technologies Used

  • Python - For backend development and data processing.
  • Flask - As the web framework to build the application.
  • Bootstrap - For responsive and modern front-end design.
  • SQL - For database management and storage.
  • JavaScript - To enhance interactivity and user experience.
  • Machine Learning - For disease prediction, personalized recommendations, and improving system performance.

Contributions

Anshika Jain has made invaluable contributions to the development of DocNow. From designing the user interface to coding the algorithms that power our recommendations, her efforts have resulted in a comprehensive and user-friendly health care solution.

Future Plans

We are continuously working on improving DocNow with new features and enhancements. Our goal is to expand our recommendations, integrate more advanced technologies, and ensure that DocNow remains at the forefront of health care innovation.
